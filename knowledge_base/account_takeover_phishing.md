# Social Media / LINE Account Takeover Phishing

Category: 網路釣魚 / 帳號盜用

Description:
Messages designed to steal the victim's LINE, Facebook, Instagram, or Apple/Google account credentials. Once the account is compromised, scammers use it to send gift-card scam messages to all of the victim's contacts — exploiting the trust of real relationships.

Common delivery methods:
- fake login page mimicking LINE, Facebook, or Apple ID
- fake "account suspended / verify now" alert with link
- fake two-factor authentication page that captures OTP in real time
- fake LINE Points / reward redemption that requires login

Specific LINE account takeover:
- 「您的LINE帳號在異地登入，請立即至 https://line-verify.xyz 確認」
- asks for LINE phone number + OTP to "restore" the account
- once taken over, uses account to send gift-card scam to contacts

Facebook / Instagram variant:
- copyright or community standards violation notice
- directs to a fake Meta login page
- steals credentials + may use ads manager to run fraudulent campaigns

Apple / Google account variant:
- fake iCloud or Google account suspension notice
- collects Apple ID / Google password + OTP
- used to make purchases or access stored payment methods

Example messages:
> 【LINE官方通知】您的帳號已在未知裝置登入，為保護帳號安全，請立即點擊 https://line-security.xyz/verify 確認。

> 您的Facebook帳號因違反社群規範已被限制，請在24小時內至 https://fb-appeal.xyz 提交申訴，否則帳號將永久停用。

> 【Apple】您的Apple ID已被停用，請至 https://apple-id-tw.xyz 重新驗證，否則裝置將被鎖定。
